## Project Objective

To improve safety by automatically detecting refrigerant gas leaks and shutting off the gas supply through a relay module, reducing the possibility of prolonged leakage and enabling faster response.

# RLD – Refrigerant Leak Detector

RLD (Refrigerant Leak Detector) is a safety-focused embedded systems project that I developed during my school years.

Many refrigerants used in air conditioning and refrigeration systems are colorless and odorless, making leaks difficult to detect without specialized equipment. In enclosed environments, significant leaks can reduce oxygen levels and create hazardous conditions.

This project continuously monitors for refrigerant gas leakage using a gas sensor. When a leak is detected above a predefined threshold, the system automatically activates a relay module to shut off the gas supply, helping reduce the risk of further leakage. It also provides an immediate warning through alert indicators.

The project was developed as a learning experience in embedded systems, sensor interfacing, and automation, and was simulated using Wokwi.
